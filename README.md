## Installation

```
$ git clone https://github.com/josephgodwinkimani/laravel-inertia-react-starter
$ cd laravel-inertia-react-starter


$ composer install
$ npm install
$ npm run dev

$ cp .env.example .env
$ php artisan key:generate
$ php artisan migrate
$ php artisan serve
```
